# Privacy Practices

Claude Usage Pace is local-only.

## Data accessed

- Visible usage percentages and reset labels from Claude's Usage settings page
- Extension preferences such as working days, working hours, compact mode, and debug mode
- Optional compact local history of usage percentages and timestamps

## Data not collected

- Claude prompts
- Claude responses
- Conversation contents
- Account identifiers
- Personal information
- Analytics events
- Browsing history outside claude.ai

## Data sharing

- No data is sold
- No data is transferred to third parties
- No data is used for advertising
- No data leaves the user's browser

## Permissions

- `storage`: saves local preferences and compact local history
- `https://claude.ai/*`: reads visible usage information and injects the pacing dashboard on Claude pages
