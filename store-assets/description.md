# Claude Usage Pace

Claude Usage Pace adds a small local-only dashboard to Claude's Usage settings page.

Claude shows how much you used. Claude Usage Pace shows whether that usage is healthy for right now.

It helps you understand:

- Current five-hour session headroom
- Weekly all-model usage pace
- Model-specific weekly limits when visible
- Whether you are under pace, on track, or at risk
- How much usage may remain unused if your pace continues
- Whether it is a good time to start a focused Claude Code task

The extension reads only the usage text already visible on Claude's settings page. It does not use a backend, does not track you, and does not send usage data anywhere.

## Limitations

Claude usage is variable and depends on model, context length, files, features, and task complexity. This extension does not estimate exact messages or tokens left. It provides percentage-based pacing guidance only.
